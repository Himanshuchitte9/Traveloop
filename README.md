# Traveloop
Traveloop is an intelligent travel planning platform that helps users create personalized multi-city itineraries, manage budgets, explore destinations and activities, organize trips, and share travel experiences seamlessly in one interactive app.

## Project Details
This project was developed based on the mobile UI designs available at [Figma](https://www.figma.com/design/9e3piHNfR5VouLDFbwsodB/Design-Mobile-App-UI).

## Running the code
1. Run `npm i` to install the dependencies.
2. Run `npm run dev` to start the development server.

## Features
- Personalized multi-city itineraries
- Budget management and expense tracking
- Destination and activity discovery
- Social travel sharing
- Packing checklists and trip journals
